# Icon Assets

This directory needs the following icon files for the PWA functionality:

## Required Icons
- `icon-192x192.png` - 192x192 pixel icon for PWA installation and smaller devices
- `icon-512x512.png` - 512x512 pixel icon for PWA installation and larger devices

## Screenshot Images (Optional)
- `screenshot-desktop.png` - Screenshot of desktop view (recommended size: 1280x800)
- `screenshot-mobile.png` - Screenshot of mobile view (recommended size: 750x1334)

You should create these icon files with:
- A simple, recognizable design representing audio transcription
- Transparent background
- Appropriate padding
- Consistent color scheme matching the application's primary color (#2196F3)

These files are referenced in the `manifest.json` file and are required for proper PWA installation.
